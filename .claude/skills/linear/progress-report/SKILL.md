# Progress Report

Generate a comprehensive progress report from issue tracker and Git data.

## When to Invoke

- Daily standup summaries
- Weekly progress reports
- Monthly retrospectives
- Sprint/cycle reviews
- Stakeholder updates

## Arguments

| Argument | Period | Linear `--since` | Git `--since` |
|----------|--------|-------------------|---------------|
| `day` or `1d` | Last 24 hours | `1d` | `1 day ago` |
| `week` or `7d` (default) | Last 7 days | `7d` | `7 days ago` |
| `month` or `30d` | Last 30 days | `30d` | `30 days ago` |
| `cycle` | Current cycle | Use cycle dates | Use cycle start |
| Custom (e.g., `14d`) | Last N days | `Nd` | `N days ago` |

Default: **7 days**

## Execution Flow

```
1. Parse time period
2. Gather data (Linear + Git + GitHub)
3. Compute metrics
4. Estimate financial value
5. Generate report
```

## Data Gathering

### Issue Tracker

```bash
# Load environment
direnv allow && eval "$(direnv export bash)"

# Stats for period
linear issues stats --team TU --since ${SINCE}

# Issues opened/closed
linear issues stats opened --team TU --since ${SINCE}
linear issues stats closed --team TU --since ${SINCE}

# Current in-progress
linear issues list --team TU --state "In Progress"
```

### Git

```bash
# Commits
git log --oneline --since="${SINCE_GIT}" --no-merges

# Commit count
git log --oneline --since="${SINCE_GIT}" --no-merges | wc -l

# Files changed
git log --since="${SINCE_GIT}" --no-merges --pretty=format: --name-only | sort -u | wc -l

# Lines added/removed
git log --since="${SINCE_GIT}" --no-merges --numstat | awk '{add+=$1; del+=$2} END {print "+"add, "-"del}'
```

### GitHub

```bash
# PRs merged
gh pr list --state merged --search "merged:>=${START_DATE}" --limit 100 --json number,title

# PR count
gh pr list --state merged --search "merged:>=${START_DATE}" --limit 100 --json number | jq length
```

## Metrics

| Metric | How to Compute |
|--------|---------------|
| Issues closed | From stats |
| Issues opened | From stats |
| Net delta | Opened - Closed |
| Commits | Git log count |
| PRs merged | GitHub PR count |
| Lines changed | Git numstat |
| Daily run rate | Closed / days |
